# QWARS.DEV

The source code of the *QWARS.DEV* blog. It contains customized build scripts, as well as its own theme for Bulma. Uses Jekyll as a static generator. You can use this code for any purpose or as a basis for your project.

## Distinctive features

- Modern and fast JS / TS compiler - ESBuild
- Configured live-reload using browser-sync
- Minimal service-worker written using Google Workbox
- Using Bulma and feather-icons

## Requirements

- Ruby `3 or higher`
- Node `16 or higher`

## Prepare environment

After cloning the project, install the necessary dependencies.

### Install Jekyll dependencies

> $ bundle

### Install node.js dependencies

> $ npm install

## Workflow

Start the development workflow, execute the npm script

> $ npm run start
